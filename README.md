🎧 Spotify Music Insights Dashboard – Power BI

A visually engaging and interactive Power BI dashboard designed to analyze Spotify music trends, song popularity, and artist performance.

📌 Project Overview

This project presents a Spotify-themed analytics dashboard built using Power BI.
It transforms raw music data into meaningful insights with a clean UI, interactive filters, and visually rich charts.

The dashboard helps users explore:

Most popular songs

Top performing artists

Popularity differences across album types

Year-wise and quarter-wise trends

Song duration, explicit content, and listening patterns
